SwimIQ branding — three short file names
========================================

Put PNGs in this folder. The app tries the short names first.

1) icon.png   (REQUIRED)
   Square app picture — 512 x 512 px.
   Login, splash, membership, web tab icon source.

2) banner.png (RECOMMENDED)
   Wide landscape strip — ~1200 x 400 px (or similar).
   Runs across PBs, Log, Goals, Video, and Passport tabs.

3) mark.png   (OPTIONAL — also accepts SwimIQ_Mark.PNG)
   Tight swimmer mark only — 512 x 512 px, mark fills ~80% of canvas.
   No wordmark, no tagline. App bar and tab banner beside the SWIMIQ wordmark.

After adding or replacing any PNG:
1. Close Chrome completely
2. Double-click KARA-SEE-UPDATES-NOW.bat or LAUNCH-CHROME.bat
   (hot reload does NOT pick up new image files)

Legacy names still work as fallbacks:
  swimiq_logo.png, swimiq_icon.png, SwimIQ_banner.PNG, swimiq_icon_mark.png

Quick copy (drag your square PNG onto DRAG-LOGO-HERE.bat):
  -> saves as assets/branding/icon.png
