SwimIQ branding — three short file names
========================================

*** LOGIN PAGE USES icon.png ONLY ***

1) icon.png   (REQUIRED — login + signup)
   Square app icon — 512 x 512 px.
   Drag your real brand icon onto COPY-LOGO.bat or SYNC-LOGO-NOW.bat.

2) banner.png (RECOMMENDED — dashboard & tab strip)
   Wide tab strip — ~1200 x 280 px.
   Shows full-width across the top of Dashboard and every tab.

3) mark.png (OPTIONAL)
   Tight swimmer mark for small slots.

OLD FILES IGNORED BY LOGIN:
  swimiq_icon.png, swimiq_logo.png — not used anymore.

After replacing icon.png: close Chrome, run KARA-SEE-UPDATES-NOW.bat, Ctrl+F5.
