============================================================
  GEMINI KEY — where it goes (simple)
============================================================

You only edit ONE Notepad file:

  Desktop\StrokeIQ\swimiq\.env

Add or keep this line (one key, no spaces around =):

  GEMINI_API_KEY=paste_the_full_key_here

WHICH KEY from Google AI Studio?
  Use EITHER one — both are Gemini keys:
    1) "Gemini API KEY"           (the one you already put in)
    2) "SwimIQ Video Analysis"    (also fine — try this if #1 fails)

  Keys can start with AIza... OR AQ....  Both are OK.

You do NOT need to open services\video_analysis\.env yourself.
When Elite starts, it COPIES the key from swimiq\.env into that file.

THEN:
  1) Save and close Notepad
  2) Close EVERY black Elite / SwimIQ server window
  3) Double-click OPEN-WORKING-APP-NOW.bat
  4) In the Elite black window, look for:
       GEMINI_API_KEY: set (coaching report enabled)
  5) Run Elite Analysis again
  6) Open the Coaching tab

If coaching still fails after that, swap in the OTHER AI Studio key
(SwimIQ Video Analysis), save, restart Elite the same way, analyze again.

Do NOT use swimiqapp.com for this test — only localhost / 127.0.0.1
