READ ME — DOUBLE CLICK ONLY
===========================

OneDrive often removes .bat files. Use the .vbs file instead.

1. Open Desktop\StrokeIQ
   If you see ANOTHER folder also named StrokeIQ inside it, open THAT one.

2. Double-click:  DOUBLE-CLICK-ME.vbs

3. An upload folder opens with:
   1-UPLOAD-THIS-TO-GODADDY.zip

4. In GoDaddy File Manager → public_html
   Upload that zip → Extract → Overwrite ALL

5. On swimiqapp.com press Ctrl+Shift+R
   You should get LOGIN (not "not connected").
