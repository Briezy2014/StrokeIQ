SWIMIQ STRIPE — BROWSER ONLY (if the black window keeps failing)
================================================================

You can turn on Stripe checkout in the Supabase website.
No Command Prompt needed.


FUNCTION 1 — create-stripe-checkout
-----------------------------------
1. Open this link:
   https://supabase.com/dashboard/project/bryurwyeosbffvfpdbv/functions

2. If you see create-stripe-checkout already, open it.
   If not, click Deploy a new function / Create function
   Name: create-stripe-checkout

3. Delete whatever code is in the editor.

4. In File Explorer open:
   OneDrive\Desktop\StrokeIQ\swimiq\supabase\functions\create-stripe-checkout\index.ts

5. Open that file with Notepad → Ctrl+A → Ctrl+C

6. Paste into the Supabase function editor → Deploy


FUNCTION 2 — stripe-webhook
---------------------------
1. Same Functions page

2. Create or open: stripe-webhook

3. IMPORTANT: turn OFF "Verify JWT" / JWT verification for this function
   (webhooks come from Stripe, not logged-in users)

4. Open in Notepad:
   OneDrive\Desktop\StrokeIQ\swimiq\supabase\functions\stripe-webhook\index.ts

5. Ctrl+A → Ctrl+C → paste into editor → Deploy


THEN — Stripe webhook
---------------------
1. Stripe Dashboard → Test mode ON
2. Developers → Webhooks → Add endpoint
3. URL:
   https://bryurwyeosbffvfpdbv.supabase.co/functions/v1/stripe-webhook
4. Events:
   checkout.session.completed
   customer.subscription.updated
   customer.subscription.deleted
5. Copy Signing secret (whsec_...)
6. Supabase → Edge Functions → Secrets
7. STRIPE_WEBHOOK_SECRET = that whsec_ value
