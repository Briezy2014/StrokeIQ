SwimIQ - what to double-click (Desktop\StrokeIQ)

IMPORTANT: Double-click the .bat file (gear/window icon), NOT an Edge/browser icon.
If Windows opens Edge, right-click the file -> Open with -> Command Prompt,
or right-click -> Run as administrator is not needed - just Open.

1) FIRST (when files are missing):
   CLICK-ME-FIRST.bat
   or GET-LATEST-FIXED-APP.bat

2) Elite analysis / "storage download" error:
   FIX-STORAGE.bat
   Leave the Elite server window OPEN.
   Health page must show storage_download_configured: true

3) Put Flutter app on https://swimiqapp.com:
   PUBLISH-SWIMIQAPP-COM.bat
   Then upload the zip to GoDaddy public_html and Extract.

If pip shows getaddrinfo failed / cannot reach pypi.org:
   That is Wi-Fi/DNS. The Elite starter now skips pip when packages
   are already installed, so you can still start the server offline.

Do NOT use Streamlit for coaches.
